puredate.js

Now is very easy manipulate date without time in javascript.

The 'puredate.js' library serves to manipulate simple dates in javascript. This is very useful when you do not need worry about time and timezone. The PureDate object is ideal to manipulate birthdays and other dates that are recorded in documents.

Download now from github: https://github.com/smarcelobr/puredate.js

Some examples of their use:

  Today

How to get current date

Code:

 var today = PureDate.today();
 document.write('Today is: '+today.year+'-'+today.month+'-'+today.day);

Result:

  addMonths(...)

Code:

 var today = PureDate.today();
 var nextMonth = today.addMonths(1);
 document.write('Today is: '+today.year+'-'+today.month+'-'+today.day);
 document.write('\nNext Month is: '+nextMonth.year+'-'+nextMonth.month+'-'+nextMonth.day);

Result:

Note: puredate objects are immutable!

  getLastDayOfMonth(...)

Code:

 var today = PureDate.today();
 var lastDayofMonth = today.getLastDayOfMonth();
 document.write('Today is: '+today.year+'-'+today.month+'-'+today.day);
 document.write('\nLast day of this month is: '+lastDayofMonth.year+'-'+lastDayofMonth.month+'-'+lastDayofMonth.day);

Result:

Note: puredate objects are immutable!

  addDays(...)

Code:

 var today = PureDate.today();
 var nextMonth = today.addDays(5);
 document.write('Today is: '+today.year+'-'+today.month+'-'+today.day);
 document.write('\n5 days ahead is: '+nextMonth.year+'-'+nextMonth.month+'-'+nextMonth.day);

Result:

Note: puredate objects are immutable!

  Recovering the day of week

Code:

 var today = PureDate.today();
 var dayOfWeek = today.dayOfWeek();
 document.write('Today is: '+today.year+'-'+today.month+'-'+today.day);
 document.write('\nDay of Week: '+dayOfWeek);

Result:

Note: 0 is Sunday, 1 is Monday, ... and 6 is Saturday

  Setting a specific date

Code:

 var christmas = PureDate.ymd(2015, PureDate.DECEMBER, 25);
 document.write('2015 Christmas is: '+christmas.year+'-'+christmas.month+'-'+christmas.day);

Result:

Note: 1 is January, 2 is February, ... and 12 is December

  Difference between dates in days

How many days until christmas?

Code:

        var today = PureDate.today();
        var nextChristmas = PureDate.ymd(today.year, PureDate.DECEMBER, 25);
        if (nextChristmas.gdate<today.gdate) { // true = the Christmas this year is over...
            nextChristmas = PureDate.ymd(today.year+1, PureDate.DECEMBER, 25);
        }
        var differenceInDays = nextChristmas.gdate-today.gdate;
        document.write('There are only '+differenceInDays+ ' days until christmas!');

Result:

Note: 'gdate' property would be the number of days since March, day 1 of year 0000 if Gregorian calendar was valid at that time. Therefore, the difference between 'gdate' is only valid for dates that fall inside the Gregorian calendar. No check is performed, since many countries have adopted the Gregorian calendar at different times.
