puredate.js Examples

Today

Code:

 var today = PureDate();
 document.write('Today is: '+today.year+'-'+today.month+'-'+today.day);

addMonths(...)

Code:

 var today = PureDate();
 var nextMonth = today.addMonths(1);
 document.write('Today is: '+today.year+'-'+today.month+'-'+today.day);
 document.write('\nNext Month is: '+nextMonth.year+'-'+nextMonth.month+'-'+nextMonth.day);

Result:

Note: puredate objects are immutable!

getLastDayOfMonth(...)

Code:

 var today = PureDate();
 var lastDayofMonth = today.getLastDayOfMonth();
 document.write('Today is: '+today.year+'-'+today.month+'-'+today.day);
 document.write('\nLast day of this month is: '+lastDayofMonth.year+'-'+lastDayofMonth.month+'-'+lastDayofMonth.day);
